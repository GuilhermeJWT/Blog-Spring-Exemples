Como baixar o Java no Windows?

Olá Seres Java, hoje iremos dar inicio ao processo de instalação do Java em nossa máquina, para podermos dar continuidade em nossos conteúdos, chega de enrolação e bora la! 1º Passo - Saber qual é a arquitetura do seu computador, no caso iremos utilizar o Windows 10, se você não sabe se é 64 ou 32 bits, faça os seguintes passos:

Painel de Controle > Sistema e Segurança > Sistema

Pronto, descobrimos que é 64 bits!

2º Passo - ir no site da Oracle ( Empresa dona do Java ) e baixar o JRE ( Java Runtime Environment ) que é para executar programas feitos em Java, irei disponibilizar o link da página para download:

Página de Download - Oracle

Ao executarmos ira abrir uma tela de instalação!

Finalizamos a primeira parte!

Ainda falta mais um passo, mais estamos quase no final!

Próximo Passo - ir no site da Oracle e ir na guia de downloads, em seguida procurar por JDK, (Java Development Kit ) que é o kit de desenvolvimento para criar aplicações na linguagem Java, irei disponibilizar o link da página para download:

Download JDK

Após isso entraremos nas opções de download de cada Sistema Operacional, como a minha é Windows 64 bits que vimos no ( 1º Passo ), irei baixar correspondente ao meu S.O, se a de vocês forem outra, baixem conforme a de vocês.

Executem o ícone do Java, semelhante ao passo 2!

Por padrão deixe sempre no diretório C:\Program Files\Java\

Feito todos os passos a JDK e JRE estaram no diretório anterior:

Agora vamos testar se está funcionando a nossa instalação?

Abra seu CMD - Tecla Windows + CMD e digite java com 'j' minúsculo

Perfeito!

Agora vamos digitar 'java -version'

Prontinho! finalizamos nossas instalações e o Java já está instalado em nosso PC \O/
